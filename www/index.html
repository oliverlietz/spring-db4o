Features
The features included in the 0.1 release are:
  • Db4oTemplate and Callback
  • Transaction Manager
  • FactoryBeans for configuring db4o
  • 2 examples available on the CVS
Note that you can find the latest features on the CVS.

Build Instructions
db4o-spring distribution comes complete with binaries and source code. To build the source code yourself for all modules simply run ant dist from the root folder. You can run the test suite on your local machine using ant tests.

Usage Instructions
See the reference manual (PDF and HTML format) found in docs/reference for more information and javadocs. If you have any questions please post on our mailing lists or forum.

Participation
The best way to participate at this point is to let us know what features you would like to see in db4o-spring. If you have a module of your own then let us know on the mailing list, forum or post a feature request on our issue tracker.
